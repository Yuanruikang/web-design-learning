块级元素的特点：需要背诵

  1. 总是从新开始
  2. 高度，行高、外边距以及内边距都可以控制
  3. 宽度默认是容器的100%
  4. 可以容纳内联元素和其他块元素

行内元素的特点：需要背诵

  1. 和相邻行内元素在一行上
  2. 高、宽无效，但水平方向的padding和margin可以设定，垂直方向的无效
  3. 宽度默认是它本身内容的宽度
  4. 行内元素只能容纳文本或其他行内元素，不能放块状元素。（a 特殊，a里面不能再放a）
  5. 只有文字才能组成段落，因此p里面不能放块级元素，同理还有h1 h2 dt

行内块元素的特点，如：

  1. 和相邻行内元素（行内块）在一行上，但是之间会有空白缝隙
  2. 默认宽度就是它本身内容的宽度
  3. 高度、行高、外边距以及内边距都可以控制
标签显示模式转换 display 块状->行内: display:inline 行内->块状：display：block 块、行内元素转换为行内块：display：inline-block
